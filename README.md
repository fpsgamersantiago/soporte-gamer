
# 🎮 Soporte Gamer - Mi Página Web

Repositorio de mi página web de soporte técnico especializado en PCs y notebooks gamer.

## 📋 Descripción

Sitio web profesional para publicitar servicios de:
- 🧹 Limpieza y reemplazo de pasta térmica
- ⚙️ Optimización térmica (undervolt, airflow)
- 🔧 Diagnóstico y upgrades de hardware

## 🚀 Características

### ✅ Optimizaciones Implementadas

- **Meta Tags SEO Mejorados**: Open Graph, Twitter Cards, descripción extendida
- **Accesibilidad**: ARIA labels, estructura semántica HTML5, navegación accesible
- **CSS Limpio**: Eliminadas duplicaciones, uso de variables CSS, responsive design
- **JavaScript Mejorado**: 
  - Scroll effect en navbar
  - Smooth scroll para links
  - Lazy loading para imágenes
  - Validación de formularios
  - Event listeners optimizados (passive listeners)
- **Páginas Completas**: 
  - `index.html` - Inicio
  - `services.html` - Detalle de servicios
  - `contact.html` - Formulario de contacto y FAQ
- **Mobile First**: Diseño responsivo en todos los dispositivos

### 🛠 Tecnologías

- **HTML5** - Estructura semántica
- **Tailwind CSS** - Framework de utilidades
- **CSS Custom** - Variables y personalización
- **JavaScript Vanilla** - Sin dependencias
- **Google Fonts** - Tipografía Orbitron
